# NUJUM - AI Prediction Analysis System
## Demo Presentation Content (10 Slides - Two Modules Only)

---

## Slide 1: Title Slide
**NUJUM - AI Prediction Analysis System**
*Comprehensive AI-Powered Prediction and Social Media Analysis Platform*

---

## Slide 2: System Overview
**What is NUJUM?**
- AI-powered prediction analysis system for any topic or domain
- Social media intelligence platform for professional assessment
- Two main modules: Prediction Analysis & Social Media Analysis

**Key Features:**
- Advanced AI analysis using Google Gemini and ChatGPT
- Multi-platform social media analysis (Facebook, Instagram, TikTok, X/Twitter)
- File upload support (PDF, Excel, CSV, TXT)
- Web scraping from source URLs
- Export capabilities (PDF reports)
- Real-time analytics dashboard

---

## Slide 3: Prediction Analysis - Dashboard
**AI Prediction Analysis Dashboard**

**What this page does:**
- Lists all user's prediction analyses in a table/card view
- Shows system information and capabilities
- Displays analysis statistics (Completed, Processing, Failed counts)
- Provides quick actions: View Details, Export PDF, Delete

**Key Elements:**
- **System Information Card**: Explains AI-powered analysis capabilities
- **Recent Predictions Table**: Shows topic, source URLs, prediction horizon, status, confidence score, date/time
- **Status Indicators**: Color-coded badges (Completed = green, Processing = yellow, Failed = red)
- **Confidence Score**: Visual progress bar showing prediction confidence percentage
- **Action Buttons**: View Details, Export PDF (for completed), Delete
- **Pagination**: For browsing through multiple analyses

---

## Slide 4: Prediction Analysis - Create Form
**New Prediction Analysis Form**

**What this page does:**
- Form to create a new AI-powered prediction analysis
- Collects all necessary information for comprehensive analysis
- Validates and processes input data before submission

**Key Form Fields:**
- **Report Title**: Required field for naming the analysis
- **Prediction Period**: Dropdown selection (Next Two Days, Two Weeks, Month, 3/6/12 Months, 2 Years)
- **Target Audience/Subject**: Optional field for specific focus area
- **Issue Description**: Large text area for detailed topic description (required)
- **Additional Resources Section** (Collapsible):
  - **Source URLs**: Add multiple web URLs for context (with validation feature)
  - **File Uploads**: Support for PDF, Excel, CSV, TXT files (max 10MB per file)

**Special Features:**
- URL validation button to check accessibility before submission
- File preview with remove option
- Progress modal during processing (shows reading URLs, analyzing data, generating predictions)
- Tooltips with helpful information for each field

## Slide 5: Prediction Analysis - Results View
**Detailed Prediction Analysis Results**

**What this page does:**
- Displays complete AI-generated prediction analysis
- Shows structured report with multiple sections
- Provides export functionality

**Key Sections Displayed:**
- **Executive Summary**: High-level overview of key insights
- **Current Situation Analysis**: Assessment of present circumstances
- **Key Influencing Factors**: Important elements affecting predictions
- **Future Predictions**: Multi-horizon forecasts (short-term, medium-term, long-term)
- **Policy Implications**: Strategic impact analysis
- **Risk Assessment**: Comprehensive risk identification with probability and impact
- **Strategic Recommendations**: Actionable guidance and next steps
- **Model Information**: AI model used, processing time, token usage, cost

**Actions Available:**
- Export to PDF button
- View model information
- Delete analysis

## Slide 6: Prediction Analysis - History
**Complete Analysis History**

**What this page does:**
- Shows all prediction analyses with advanced filtering and search
- Provides comprehensive statistics and overview
- Allows bulk management of analyses

**Key Features:**
- **History Overview Stats**: Total, Completed, Processing, Failed counts (collapsible section)
- **Search & Filter Section** (Collapsible):
  - Search by topic or target
  - Filter by status (Pending, Processing, Completed, Failed, etc.)
  - Date range filters (From Date, To Date)
  - Prediction horizon filter
  - Sort options
- **Analysis List**: Table/card view with all analyses
- **Quick Actions**: View, Export, Delete for each analysis
- **Pagination**: Navigate through large result sets

## Slide 7: Prediction Analysis - Analytics
**Personal Analytics Dashboard**

**What this page does:**
- Tracks user's prediction analysis usage, costs, and performance metrics
- Provides insights into usage patterns and trends
- Helps monitor resource consumption

**Key Metrics Displayed:**
- **Key Metrics Cards**: Total Analyses, Total Tokens, Total Cost, Success Rate
- **Performance Metrics**: Average Processing Time, Average Cost per Analysis, Average Tokens per Analysis, Analysis Type Breakdown
- **Visualizations**: 
  - Prediction Period Usage breakdown
  - Daily Analysis Count chart (bar graph)
  - Daily Cost Trend chart (bar graph)
- **Date Range Selector**: Filter analytics by custom date range
